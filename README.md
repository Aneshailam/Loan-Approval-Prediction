# Loan Approval Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a loan application will be approved or not using Machine Learning classification algorithms.

## 🎯 Objective
To automate loan approval decisions and provide explainable predictions using SHAP.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- SHAP
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Features
- Data Cleaning
- Feature Engineering
- Model Training
- Model Evaluation
- Explainable AI (SHAP)

## 📈 Outcome
Built a classification model to predict loan approval and improve decision-making transparency.
